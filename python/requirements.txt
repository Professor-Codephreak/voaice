# voaice Python speech tools — OPTIONAL backends. None are required by the voaice
# core (torch-free, zero-Python JS DSP). The easiest install is the venv script:
#
#     ./install.sh              # light profile — pyttsx3 + vosk (offline, no torch)
#     ./install.sh --online     # + gTTS + SpeechRecognition (need network)
#     ./install.sh --full       # + Coqui TTS + openai-whisper (pull torch, heavy)
#
# Or install a profile straight into your own environment:
#     pip install -r requirements-light.txt
#
# Profiles: requirements-light.txt · requirements-online.txt · requirements-full.txt
